Date | Fechas

Pipes para mostrar fechas, algunos ejemplos:

  • Fecha sin formato:
  • {{ today }}
  • Mes letra/numero: {{ today | date:'MMMM' }} / {{ today | date:'MM' }}
  • Fecha MMMM, yy, dd: {{ today | date:'MMMM, yyyy, dd' }}
  • Fecha dd/MM/yy: {{ today | date:'dd/MM/yy' }}
  • Fecha | date: {{ today | date }}
  • Fecha | date'sort': {{ today | date:'sort' }}
  • Fecha | date:'long': {{ today| date:'long' }}
  • Fecha Ingles USA(date:'medium':'GTM-0':'en-US'): {{ today | date:'medium':'GTM-0':'en-US' }}
  • Fecha Ingles GB(date:'medium':'GTM-5':'en-GB'): {{ today | date:'medium':'GTM-5':'en-GB' }}
  • Fecha Frances Canada {{ today| date:'medium':'GTM-1':'fr-CA' }}
  • Fecha Frances Frances long: {{ today| date:'long':'GTM-5':'fr' }}
