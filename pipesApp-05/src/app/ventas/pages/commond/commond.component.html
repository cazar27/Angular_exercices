Date | Fechas

Pipes para mostrar fechas, algunos ejemplos:

  • Fecha sin formato
  • {{ today }}
  • Fecha | date {{ today | date }}
  • Fecha | date'sort' {{ today | date:'sort' }}
  • Fecha | date:'long' {{ today| date:'long' }}
  • Fecha dd/MM/yy {{ today | date:'dd/MM/yy' }}
  • Mes Ingles {{ today | date:'MMMM' }} - {{ today | date:'MM' }}
  • Fecha MMMM, yy, dd {{ today | date:'MMMM, yyyy, dd' }}
